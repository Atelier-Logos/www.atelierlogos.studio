# Getting Started

Welcome to Ferropipe! This guide helps you query Rust crates via our API in minutes.

## Prerequisites

- Basic understanding of REST APIs
- An API client (e.g. `curl` or Postman)

## 1. Get Your API Key

1. [Book an intro call](https://cal.com/team/atelier-logos/ferropipe-intro)
2. Choose your tier ([see pricing](/docs/pricing))
3. Pay the invoice and receive your API key
4. Keep your key secure (don't commit it)

## 2. Make Your First Request

Let's start with a simple request to get information about the popular serde crate:

### Using Curl

```bash
curl -X POST http://platform.atelierlogos.studio/analyze \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "name": "serde",
    "version": "1.0.0",
    "api_key": "YOUR_API_KEY",
  }'
```

The response should be a very large JSON blob that can be used for anything from documentation, LLM context, or security analysis.
