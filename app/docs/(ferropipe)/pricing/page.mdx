### Plan Recommendations

- **Basic Tier**: Perfect for personal projects, learning Rust, or experimenting with the API
- **Full Tier**: Ideal for production applications, documentation sites, and developer tools
- **Deep Tier**: Best for enterprise applications, AI training data, and comprehensive analysis platforms

## Usage Examples by Tier

### Basic Tier Use Cases

**Personal Learning Projects**
```javascript
// Get basic information about a crate
const response = await fetch('https://www.atelierlogos.studio/api/ferropipe/crate/serde', {
  headers: { 'x-api-key': 'your-api-key' }
})

const data = await response.json()
console.log(`${data.package_name}: ${data.description}`)
```

**Simple Documentation Lookup**
- Check crate versions and compatibility
- Read basic documentation
- Verify license information
- Access repository links

---

### Full Tier Use Cases

**Documentation Website Generator**
```javascript
// Get comprehensive module documentation
const response = await fetch('https://www.atelierlogos.studio/api/ferropipe/crate/tokio?depth=full&examples=true', {
  headers: { 'x-api-key': 'your-api-key' }
})

const data = await response.json()

// Generate documentation pages for each module
Object.entries(data.key_modules).forEach(([moduleName, moduleData]) => {
  generateDocPage(moduleName, moduleData)
})
```

**Developer Tools Integration**
- IDE extensions with rich documentation
- API documentation generators
- Code analysis tools
- Dependency management systems

## Rate Limiting Details

### How Rate Limits Work

Rate limits are enforced per API key on a rolling hour window. Each plan includes:

- **Request counting**: Every API call counts toward your hourly limit
- **Reset timing**: Limits reset on a rolling basis (not fixed hourly windows)
- **Overage handling**: Requests exceeding the limit receive a `429 Too Many Requests` response
- **Rate limit headers**: All responses include current usage information

### Rate Limit Headers

```http
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 847
X-RateLimit-Reset: 1640995200
```
